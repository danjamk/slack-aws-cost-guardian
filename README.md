# Slack AWS Cost Guardian

> AI-powered AWS cost monitoring with Slack alerts

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Python 3.12](https://img.shields.io/badge/python-3.12-blue.svg)](https://www.python.org/downloads/)
[![AWS CDK](https://img.shields.io/badge/AWS_CDK-v2-orange.svg)](https://aws.amazon.com/cdk/)

Detect spending anomalies, track budgets, get intelligent analysis delivered directly to Slack.

## Features

- **Daily & Weekly Cost Reports** - Automated summaries with budget tracking
- **AI-Powered Anomaly Detection** - Claude analyzes unusual spending patterns
- **Interactive Slack Alerts** - Acknowledge expected costs directly from Slack
- **Historical Context** - Learns from your feedback to reduce false positives
